# Multi Launch Example

## Description

This example demonstrates how to combine multiple launch configurations into a single launch script. It shows the advanced usage of dynamically loading node configurations from different launch files and merging them together.

## Structure

```bash
multi_launch/
├── launch.py  # multi-launch script
└── README.md  # this file
```

## Dependencies

### Hardware

None

### Environment

None

## Usage

- Assuming you have installed the library from source code, and your `working directory` is `hex_zmq_servers/examples/adv/multi_launch`, you can run the example by:

    ```bash
    source ../../../.venv/bin/activate
    python3 launch.py
    ```
